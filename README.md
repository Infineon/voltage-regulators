# voltage-regulators Repository

This repository hosts Linux device drivers for Infineon's Point-of-Load (POL) Voltage Regulators and Digital Multiphase Controllers.

Release Contents includes the following components:

- **README:** This file provides an overview of the repository and its contents.

- **POL Folder:** Linux driver for Infineon POL voltage regulators:
   - TDA38725A/40A iPOL:
   0001-hwmon-pmbus-tda38740a-Add-driver-for-Infineon-TDA387.patch
