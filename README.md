# voltage-regulators
Voltage Regulators Repository
This repository hosts Linux device drivers for Infineon Point-of-Load (POL) Voltage Regulators and Digital Multiphase Controllers.

Release Contents
This release includes the following components:

1. README: This file provides an overview of the repository and its contents.

2. POL Folder: Linux driver for Infineon POL voltage regulators, including:
   a. TDA38725A/40A iPOL

3. Multiphase Folder: Linux driver for Infineon digital multiphase controllers, including:
   a. XDPE19284C and XDPE192C4B
